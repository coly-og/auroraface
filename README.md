# AuroraFace - E-commerce de Cosméticos Premium

## 🌟 Sobre

AuroraFace é um e-commerce moderno e profissional para venda de produtos de beleza e cosméticos premium. Desenvolvido com design elegante e sofisticado inspirado em marcas de luxo como Océane.

## ✨ Características

- ✅ **Autenticação de Usuários**: Registro e login seguro com criptografia bcryptjs
- ✅ **Catálogo de Produtos**: Exibição elegante de 6 produtos premium com imagens
- ✅ **Carrinho de Compras**: Adicionar, remover e gerenciar quantidade
- ✅ **Múltiplas Moedas**: Suporte para BRL, USD e EUR com conversão automática
- ✅ **Pagamento PIX**: QR Code gerado automaticamente com segurança
- ✅ **Design Responsivo**: Funciona em desktop, tablet e mobile
- ✅ **Endereço de Entrega**: Cadastro obrigatório durante registro
- ✅ **Design Premium**: Tons de ouro e terra inspirados em Océane

## 🚀 Como Rodar Localmente

### Pré-requisitos
- Node.js (versão 14+)
- npm ou yarn

### Instalação

```bash
# 1. Clone o repositório
git clone https://github.com/coly-og/auroraface.git
cd auroraface

# 2. Instale as dependências
npm install

# 3. Configure as variáveis de ambiente
cp .env.example .env

# 4. Inicie o servidor
npm start
# ou em modo desenvolvimento:
npm run dev
